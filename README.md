# CyberBot - Cybersecurity Awareness Chatbot

A WPF-based cybersecurity awareness chatbot that educates users about online safety topics through an interactive GUI.

  # Features Implemented
  ## Part 1- Console Foundation
  - Text-based chatbot with keyword recognition
  - ACSII art displayed in the header on startup
  - Voice-greeting launched
 
  ## Part 2- WPF GUI
  -WPF GUI with header, chat display, input box, and send button
  -Memory store to remember user name and favourite topic
  -Sentiment detector for user emotional tone
  -Personalised responses based on user context
  -Keyword responder covering : passwords, phishing, malware, etc

  ## Part 3- Extended features
  -Task Assistant: Add, view, complete and delete cybersecurity tasks
  -JSON storage: Tasks saved to tasks.json using Newtonsoft.Json(full crud)
  -Cybersecurity quiz: 10+ questions across the topics with immediate feedback
  -NLP Simuation:Intent detection using string.Contains for natural phrasing
  -Activity Log: Timestapped log of all significant actions with Show More feature
